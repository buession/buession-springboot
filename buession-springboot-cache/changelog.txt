Buession SpringBoot Cache Changelog
===========================


Changes in version 1.3.2 (2021-11-10)
-------------------------------------
*【变化】ConfigurationProperties 通过 Auto Configuration 构造函数注入
*【变化】统一化 ConfigurationProperties 属性命名


Changes in version 1.3.1 (2021-07-27)
-------------------------------------
*【修复】spring.redis.key-prefix 配置未生效的 BUG


Changes in version 1.3.0 (2021-07-12)
-------------------------------------
*【变化】移除静态方法上的 final 关键字
*【变化】AutoConfiguration 类 @Configuration proxyBeanMethods 为 false
*【变化】Redis 连接 Bean Configuration 从 RedisConfiguration 拆分为 JedisConnectionConfiguration
*【变化】Redis host 和 Uri 都没有定义值时，将抛出异常
*【变化】连接池属性 minEvictableIdleTime、numTestsPerEvictionRun、testWhileIdle 默认值修改为何 JedisPoolConfig 保持一致
*【变化】移除 commons-logging
*【管理】逐步规范化 changelog 和 测试用例
